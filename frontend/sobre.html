Sobre o Projeto

Entenda como funciona o classificador de e-mails com Inteligência Artificial.

O que é o projeto?

O Classificador de E-mails é um sistema que utiliza Inteligência Artificial para analisar mensagens recebidas e identificar se elas são Spam ou Não Spam.

Objetivo

O objetivo do projeto é ajudar o usuário a manter sua caixa de entrada mais segura e organizada, filtrando mensagens suspeitas e facilitando a identificação de possíveis e-mails indesejados.

Como funciona

1. Coleta de e-mails

O sistema recebe ou coleta os dados principais do e-mail.

2. Análise com IA

A IA analisa remetente, assunto e conteúdo da mensagem.

3. Classificação

O e-mail é classificado como Spam ou Não Spam.

4. Histórico

O resultado é salvo e exibido no painel principal.

Tecnologias utilizadas

Python Flask Scikit-learn TF-IDF SQLite HTML CSS JavaScript
