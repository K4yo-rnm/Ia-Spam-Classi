# Histórico de Versões

## v1.0

Primeira versão funcional do projeto.

### Funcionalidades

- Modelo de IA treinado para classificar mensagens como spam ou não spam
- Backend Flask com rota `/prever`
- Frontend conectado com o backend
- Testes realizados via Thunder Client
- Estrutura inicial organizada em IA, Backend e Frontend

### Limitações

- A análise ainda é feita manualmente digitando o texto
- Ainda não possui login
- Ainda não possui integração com Gmail
- Ainda não salva histórico de mensagens
- Ainda não possui sistema de feedback
